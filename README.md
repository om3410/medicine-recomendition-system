# medicine-recomendition-system
Overview
This project is a machine learning–driven web application that recommends medicines based on user-input symptoms. It aims to assist patients and healthcare workers by providing quick, data-backed suggestions for treatment, precautions, and recovery plans. Built using Python, Flask, and Scikit-learn, the system demonstrates how AI can support healthcare accessibility and decision-making.

🎯 Purpose
To create a lightweight, accessible tool that:
- Predicts diseases from symptoms using ML classification
- Recommends medicines, precautions, and lifestyle changes
- Can be extended to support regional languages and ethical AI filters

  
📊 Dataset Structure
- symptom_disease.csv: Maps symptoms to diseases
- medicine.csv: Lists medicines per disease
- precautions.csv: Safety tips and lifestyle suggestions
- diet.csv: Recommended foods and restrictions
- workout.csv: Recovery exercises

🩺 What Is This Project About?

This is a smart web app that helps people figure out what medicine they might need based on the symptoms they enter. It’s like a mini doctor assistant that gives suggestions for:
- What disease the symptoms might point to
- Which medicines could help
- What precautions to take
- What food to eat or avoid
- Simple workouts to support recovery

It’s built using Python and Flask, with machine learning doing the brainwork behind the scenes.


💡 Why I Built This

In many places, especially rural areas, people don’t always have quick access to doctors. This system gives basic guidance that can help them take the right steps early. It’s not a replacement for a doctor—but it’s a helpful tool when one isn’t nearby.
